## 3. 效益計算公式

### 3.1 人力成本節省

**現況分析**：

| 項目 | 數值 | 來源 |
|------|------|------|
| 週期時間量測人力 | 2 人 | 現場訪談 |
| 每週量測工時 | 8 小時/人 | 現場訪談 |
| 年度量測工時 | 2 × 8 × 52 = 832 小時 | 計算 |
| 作業員時薪 | 250 元 | 公司薪資標準 |
| 年度人力成本 | 832 × 250 = 208,000 | 計算 |

**導入後預估**：

| 項目 | 數值 | 計算方式 |
|------|------|---------|
| AI 自動化比例 | 80% | 保守估計 |
| 仍需人工校正 | 20% | |
| 節省工時 | 832 × 0.8 = 665.6 小時 | |
| 年度節省金額 | 665.6 × 250 = **166,400** | |

**加計 IE 工程師工時節省**：

```
IE 工程師原本每週花 4 小時整理數據
- 年度 = 4 × 52 = 208 小時
- IE 工程師時薪 = 400 元
- 節省 = 208 × 400 × 0.8 = 66,560

總人力節省 = 作業員工時 + IE 工時
          = 166,400 + 66,560
          = 232,960
          ≈ 250,000 (保守取整)
```

**注意**：不計入「轉為生產性工作的產值」，因為需視實際產能需求而定

**最終數字**：人力成本節省 = **NT$250,000/年** (保守估計)

---

### 3.2 品質成本降低

**計算假設**：

| 項目 | 數值 | 來源 |
|------|------|------|
| 年度不良批次 | 50 批 | 品保數據 |
| 每批處理成本 | 20,000 元 | 品保數據 |
| 年度品質成本 | 50 × 20,000 = 1,000,000 | 計算 |

**導入後預估**：

```
即時監控可提早發現異常的比例 = 15%
(保守估計，業界經驗為 20-40%，但第一年系統仍在磨合)

品質成本降低 = 1,000,000 × 0.15 = 150,000
```

**注意**：實際效益需視品保部門提供的歷史數據而定

**最終數字**：品質成本降低 = **NT$150,000/年** (保守估計)

---

### 3.3 效率提升產值

**計算假設**：

| 項目 | 數值 | 來源 |
|------|------|------|
| 年度產值 | 50,000,000 | 財務數據 |
| 目前製程效率 | 85% | IE 數據 |
| 數據驅動改善潛力 | 3-5% | 業界經驗 [12] |

**計算公式**：

```
效率提升比例 = 2% (第一年保守估計)
產值提升 = 年度產值 × 效率提升比例
        = 50,000,000 × 0.02
        = 1,000,000

考慮實現率 (第一年預估只達成 20%)：
實際產值提升 = 1,000,000 × 0.20 = 200,000
```

**注意**：效率提升需要累積足夠數據後才能發揮價值，第一年效益有限

**參考文獻**：
- [12] McKinsey "AI in Manufacturing" Report 2023

**最終數字**：效率提升產值 = **NT$200,000/年** (第一年保守估計)

---

### 3.4 ROI 計算

**投資報酬率公式 (保守估計)**：

```
第一年效益 = 人力節省 + 品質成本降低 + 效率提升
          = 250,000 + 150,000 + 200,000
          = 600,000

第二年起效益 (系統成熟後，效益提升 50%)：
          = 600,000 × 1.5 = 900,000

投資成本 = 795,000

投資回收期 = 投資成本 ÷ 第一年效益
          = 795,000 ÷ 600,000
          = 1.33 年
          = 16 個月

3 年 ROI = (3 年效益 - 投資成本) ÷ 投資成本 × 100%
        = (600,000 + 900,000 + 900,000 - 795,000) ÷ 795,000 × 100%
        = (2,400,000 - 795,000) ÷ 795,000 × 100%
        = 1,605,000 ÷ 795,000 × 100%
        = 202%
```

**說明**：採用保守估計，第一年為系統導入期，效益有限；第二年起系統成熟，效益提升

---

### 3.5 NPV 計算

**淨現值公式 (保守估計)**：

```
NPV = Σ (CFt ÷ (1+r)^t) - C0

其中：
- CF1 = 第 1 年現金流 = 600,000
- CF2 = 第 2 年現金流 = 900,000
- CF3 = 第 3 年現金流 = 900,000
- r = 折現率 (5%)
- C0 = 初始投資 = 795,000

NPV = (600,000 ÷ 1.05) + (900,000 ÷ 1.05²) + (900,000 ÷ 1.05³) - 795,000
    = 571,429 + 816,327 + 777,454 - 795,000
    = 2,165,210 - 795,000
    = 1,370,210
    ≈ 1,370,000
```

**說明**：即使採用保守估計，NPV 仍為正值，表示投資具經濟效益

### 5.3 敏感度分析

```
情境分析 (以第一年效益 600,000 為基準)：

樂觀情境 (效益 +50%)：
- 第一年效益 = 600,000 × 1.5 = 900,000
- 投資回收期 = 795,000 ÷ 900,000 = 10.6 個月

基準情境：
- 第一年效益 = 600,000
- 投資回收期 = 795,000 ÷ 600,000 = 16 個月

保守情境 (效益 -30%)：
- 第一年效益 = 600,000 × 0.7 = 420,000
- 投資回收期 = 795,000 ÷ 420,000 = 22.7 個月

悲觀情境 (效益 -50%)：
- 第一年效益 = 600,000 × 0.5 = 300,000
- 投資回收期 = 795,000 ÷ 300,000 = 31.8 個月

結論：
- 基準情境下，約 16 個月回收
- 即使在悲觀情境下，仍可在 3 年內回收
- 考量第 2 年起效益提升，實際回收可能更快
```
