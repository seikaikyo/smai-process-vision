# 專案概述

## 專案目標

建置 **製程視覺 AI 分析系統**

- 即時影像 AI 推論（邊緣運算）
- 製程動作自動拆解
- 週期時間分析與統計學習
- 與 WISE-IoTSuite 整合

---

## 現況痛點

| 痛點 | 影響 |
|------|------|
| 人工秒錶計時 | 每週 8 小時人力 |
| 數據準確度低 | 人為誤差 15% |
| 無法即時監控 | 異常發現延遲 |
| 數據孤島 | 難以整合分析 |

## 分析功能

- **自動分段**：AI 識別取料/組裝/檢查等
- **時間軸編輯**：拖拉調整分段邊界
- **學習統計**：平均值、標準差、趨勢
- **改善建議**：自動建議標準時間
